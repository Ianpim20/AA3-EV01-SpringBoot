# AA3-EV01 - Modulo Clientes con Spring Boot

Proyecto desarrollado como evidencia AA3-EV01 del componente
"Frameworks para construcción de aplicaciones con JAVA".

## Tecnologías usadas
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Git / GitHub

## Funcionalidad
Modulo de gestión de clientes con arquitectura por capas:
- Entity
- Repository
- Service
- Controller

Endpoint disponible:
GET http://localhost:8080/clientes
